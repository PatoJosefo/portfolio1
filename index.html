My Projects

Export Statistics of São Paulo Municipalities (API)

I am currently engaged in a Scrum-based project at my academic institution, aimed at developing a website that presents critical data regarding exports from São Paulo, Brazil, to international markets. The primary objectives for the initial sprint encompassed the creation of a functional prototype using Figma, the development of responsive data visualizations utilizing Google Colab, the management of a GitHub repository, and the delivery of features as per the client's specifications.

Languages & Tools Used During The Project

Github, Figma, Google Colab, Discord, Html, CSS, Bootstrap, Jira, Slack, Python, Jupyter Notebook, Google Sheets, Git Bash and Visual Studio Code

In this project, I assumed the role of Product Owner and Scrum Master, within a team comprising eight members in the first sprint, and 6 members on the second one.

During this phase, I demonstrated proactive leadership and engagement by undertaking the following responsibilities:

1. Formulating and overseeing the Sprint and Product Backlogs, along with defining both functional and non-functional requirements.

2. Regularly updating and maintaining the project's GitHub repository, ensuring comprehensive documentation and version control.

3. Designing and developing the initial prototype using Figma, aligning with user experience best practices.

4. Serving as the primary liaison with the client, effectively communicating requirements and feedback to the development team.

5. Creating the project's site using html, css and bootstrap.

6. Creating reunion topics to engage with my team members

7. Editing a burndown chart using Excel

These efforts contributed to the successful progression of the project, aligning with the established goals and timelines.
